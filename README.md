# FakeSIM-Lsposed-module
English:

lsposed hook that works from Android 10 to Android 14 (includes MIUI 12.0 and MIUI 12.5)

Español:

Hook de lsposed que funciona desde Android 10 hasta Android 14 (incluye MIUI 12.0 y MIUI 12.5

Español: 

hook para lsposed que te permite falsificar una sim (o 2 si tienes 2 slots) dada de baja o bloqueada por pin y puk (inhabilitada) número y operadora estos 2 últimos los puedes cambiar desde /data/local/tmp/minenet con los archivos carrier, carrier_1, number y number_1 y en el caso de que uses Android 11, el icono de voLTE también será hookeado y te saldrá en la barra de estado, ROAMING esta activado por no poder forzarse ningún tipo de red

ADVERTENCIA: ESTE MÓDULO NO DA ACCESO A NINGUNA RED REAL NI A LLAMADAS REALES, LAS COBERTURAS QUE VES SON LA SEÑAL DE LAS ANTENAS PARA LLAMADAS DE EMERGENCIA/SOS, NO DARÁ ACCESO A NINGUNA RED YA QUE ESTO ES ESTÉTICO!


English:

This hook for lsposed allows you to spoof a SIM card (or two if you have two slots) that has been deactivated or blocked by PIN and PUK (disabled). 
You can change the SIM number and carrier in /data/local/tmp/minenet using the files carrier, carrier_1, number, and number_1. 
If you're using Android 11, the VoLTE icon will also be hooked and will appear in the status bar. 
Roaming is enabled because no network can be forced. 

WARNING: THIS MODULE DOES NOT PROVIDE ACCESS TO ANY REAL NETWORK OR REAL CALLS. 
THE COVERAGE YOU SEE IS JUST THE SIGNAL FROM THE ANTENNAS FOR EMERGENCY/SOS CALLS. 
IT WILL NOT PROVIDE ACCESS TO ANY NETWORK, AS THIS IS PURELY AESTHETIC!
